# Kronk CMS

This is a static site generator project.

## Guidelines
- Use the established content directory structure
- Follow the framework's conventions for layouts and templates
- Ensure all content has proper front matter
- Optimize for performance and SEO

## Quality Requirements
- Include meta titles and descriptions
- Use semantic HTML
- Ensure accessibility (WCAG 2.1 AA)
- Optimize images with proper alt text

## Available Workflows
- `/kronk-content` - Create content with guardrails
- `/kronk-audit` - Run quality audit

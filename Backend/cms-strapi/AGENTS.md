## Non-Negotiable Rules

1. **NEVER skip verification** before claiming a task is complete
2. **NEVER claim "done"** without running the build and showing output
3. **ALWAYS check SEO** when creating or editing any page
4. **ALWAYS check code quality** when modifying HTML/CSS/JS
5. **ALWAYS enforce theme consistency** when changing templates
6. **ALWAYS match writing style** of existing content when creating new content
7. **ALWAYS optimize images**: alt text, responsive srcset, proper formats

## Content Creation Workflow

1. Review existing content in `src/api/` to understand structure

4. Validate SEO: title 30-60 chars, description 120-160 chars, OG tags
